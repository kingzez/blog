#### sublime text 2.x：
具体设置：Preferences -> Settings – Default -> 搜索open_files_in_new_window，将其true 改为 false 后重启一下sublime text 2
<!--more-->
#### sublime text 3.x：
具体设置：Preferences -> Settings – User -> 添加 "open_files_in_new_window": false,
重启一下sublime text 3
